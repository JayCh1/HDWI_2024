How to use this page:

This page is especially developed for users in Alaska. Because HDWI is designed to work during daylight hours, the Alaska CFSR climatology uses the maximum HDWI from 1800, 0000, and 0600 UTC for a given day, while the GEFS forecasts use 1800, 2100, 0000, 0300, 0600, and 0900 GEFS output. All other functionality should be the same as the main HDWI page.

As noted elsewhere, all of the products on this website are still in active development and should not be considered operational. We make every attempt to keep this site updated, but outages may happen. Use these products at your own risk.
